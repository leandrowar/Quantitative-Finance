Web Scraping com R - Coletando dados da B3
=====================

 Neste vídeo mostro como construir o seu primeiro web crawler em R para coletar os dados dos Ajustes Diários e das informações de balanços das empresas listadas na B3.
 
 Utilizo a biblioteca rvest e mostro como em poucas linhas você consegue ter acesso a estes dados muito valiosos na hora de investir.
