Trading prevendo o preço do BTC com o Google Trends
=====================

 Já pensou prever o preço do Bitcoin baseado nos resultados do Google Trends?

 Neste vídeo mostro como criar um modelo matemático no Excel utilizando dos dados do Google Trends para criar um trading system diário para o Bitcoin. O resultado é de +17% de lucro já descontados os custos operacionais, contra -51% do Buy & Hold.
